\# Parmana Trust Core v1.1.0



Release Date: June 2026



Status: Open Source Developer Preview



\---



\## Summary



Parmana Trust Core introduces an open trust architecture focused on Authorization → Intent → Execution verification.



This release establishes the foundational trust model, trust artifacts, verification architecture, documentation corpus, OpenAPI specifications, and integration patterns.
