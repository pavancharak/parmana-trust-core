\# Parmana Trust Core v0.4.0 Release Notes



Release Date: 2026-06-21



Status: Released



\---



\## Summary



Parmana v0.4 delivers the Execution Trust Chain™, a verifiable infrastructure that validates authority and organizational policy before autonomous actions occur.



Parmana prevents unauthorized autonomous actions.



Parmana verifies human-defined authority and organizational policy before autonomous systems act.



\---



\## Major Deliverables



\### Decision Attestation



Implemented signed decision attestations.



\### Verification Receipts



Implemented verification receipts with policy validation.



\### Execution Tokens



Implemented execution authorization tokens.



\### Execution Records



Implemented execution recording and lineage preservation.



\### Trust Chain Retrieval



Implemented complete trust chain retrieval.



\### Transparency Log



Implemented append-only transparency logging.
