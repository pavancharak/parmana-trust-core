\# Task Registry



\## Overview



The Task Registry defines and manages tasks that may be authorized within a trust domain.



Tasks represent actions subject to policy evaluation.



\## Responsibilities



\* Register tasks

\* Resolve task definitions

\* Manage task metadata



\## Inputs



\* Task Definitions



\## Outputs



\* Task Records



\## Trust Lifecycle Position



Subject → Task → Policy
