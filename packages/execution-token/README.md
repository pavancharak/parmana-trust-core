\# Execution Token



\## Overview



Execution Tokens authorize execution following successful verification.



They establish a cryptographic linkage between verified intent and execution.



\## Responsibilities



\* Represent execution authorization

\* Preserve intent linkage

\* Support execution verification



\## Inputs



\* Verification Receipt



\## Outputs



\* Execution Token



\## Trust Lifecycle Position



Verification Receipt → Execution Token
