\# Signal Registry



\## Overview



The Signal Registry manages signals used during authorization evaluation.



Signals provide contextual evidence for decision making.



\## Responsibilities



\* Register signals

\* Resolve signal definitions

\* Support signal evaluation



\## Inputs



\* Signal Data

\* Signal Definitions



\## Outputs



\* Signal Evaluations



\## Trust Lifecycle Position



Policy → Signals → Authority Decision
