\# Attestation



\## Overview



The Attestation package generates cryptographically verifiable attestations from authorization decisions.



Attestations preserve authorization intent for downstream verification.



\## Responsibilities



\* Generate attestations

\* Preserve authorization evidence

\* Bind intent to authorization

\* Support independent verification



\## Inputs



\* Authorization Decision

\* Intent

\* Policy Metadata



\## Outputs



\* Attestation



\## Trust Lifecycle Position



Decision → Intent → Attestation
